// Message from the Coach

試合に勝つ、目標達成、選手一人一人の笑顔が沢山見たい！ でも、バドミントンには必ず勝ち負けがついてくる。 努力を続けることで、多くの夢が叶い、笑顔が増える。 その考えを大切にして、北九州市で小中学生のバドミントンの 競技力向上や普及に取り組んでいます。

Coach Nishimura

// 5 Principles

井堀バドミントンクラブの
5つの考え

01

考える力を身に付ける

自己分析→実行→失敗→また考える

「何を頑張るべきか」「今の自分に何が足りてないのか」と自己分析し、考えたことを実行。失敗しても、また考えて実行する。

02

チームを家族と考える

指導者・選手・保護者の垣根なし

選手は家族。馬鹿を言い合える環境。バドミントンだけでなく、家族のように選手一人一人の勝利をチーム全体で応援する。

03

楽しい思い出を沢山作る

プール・お祭り・潮干狩り・海遊び

バドミントンだけが全てではない。色々な人との交流、遠征・合宿・お楽しみ会。思い出が一つでも増えるチーム運営を。

04

選手自身が目標を設定する

1勝→賞状→メダル→全国へ

始めた頃は1勝で大喜び！いつの間にか「賞状が欲しい」「メダル」「優勝」「全国」へ。その目標達成を叶える指導を。

05

みんながバドミントンを好きに

親もハマる。親子でハマる。

選手だけでなく親もバドミントンを好きになる。親子で好きになることが選手の成長や活躍に繋がる(笑)

// Q & A

よくある質問

Q バドミントンが初めてでも入れますか？ +
もちろん大歓迎です。ラケットを握ったことがない子も、コーチが基礎から優しく教えます。 実際、現役選手の多くも入部時は初心者でした。
Q ラケットや道具は必要ですか？ +
体験時はラケット・シャトルを無料でお貸しします。入部後は徐々に揃えていけば大丈夫です。
Q 月謝はいくらですか？ +
月謝は 5,000円/月 です。兄弟割引もご用意しています（2人目以降割引）。
Q 練習を見学だけしたいのですが？ +
見学だけでもOKです。InstagramのDMでご連絡ください。
Q 兄弟で入れますか？ +
はい。兄弟で参加される方も多く、割引制度もあります。
Q 親の付き添いは必要ですか？ +
体験時は必ず保護者の方の付き添いをお願いします。普段の練習は、お子さんだけでも大丈夫です。
Q 大会への参加は必須ですか？ +
必須ではありません。レベルや希望に応じて、参加するかどうか相談しながら決めます。
Q 中学生になっても続けられますか？ +
はい、中学3年生まで継続できます。多くの卒業生が中学校でも活躍しています。
